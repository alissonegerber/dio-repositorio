Estrutura básica do html

Voltar ao topo

primeiro, como convenção, toda primeira ou principal pagina de um site se chama index.html

Uma página html consiste em uma estrutura de 3 partes: html, tag que inicia e encerra o código da página; cabeça (head) que serão coisas que vão carregar antes do corpo do site, como o título; e corpo que apresenta os conteúdos do site em si (body)

Tags I

Voltar ao topo

Tags são o que vem entre < e > para começar, e muitas vezes com < e /> para encerrar, por exemplo a tag html, que começa com < html> e encerra com < / html>, mas nem todas encerram dessa forma, como a tag < img / > e desempeham diversas funções, como as tags < h1> e < h2> para títulos, < body > para delimitar o corpo do site, etc.

Exemplos de tags e suas funções:

  • < i > < / i > para escrever em Itálico
  • < Strong > < / Strong > para escrever em Negrgito
  • < img / > para inserir imagens

Listas

Voltar ao topo

Existem dois tipos de listas:

  1. Ordenadas < ol >
  2. Não ordenadas < ul >

Nestas listas, para iniciar se usa as tags 'ol' ou 'ul'. Dentro destas listas é preciso usar a tag < li > para adicionar intens na lista.

Links

Voltar ao topo

Algumas formas de colocar links no site:

usando a tag < a > com o atributo 'href' para incluir o site e um nome Exemplo link

A configuração básica dessa tag é a seguinte: < a href = "link" target"_blank"

Neste caso, _blank programa a função para abrir o link em outra aba, que pode ser substituída por _self, para abrir o link na aba atual.

Adicionando title = "" podemos adicionar um texto que será exibido ao usuário quando passa o mouse em cima do link, confira no link anterior, a frase "O socialismo cresce"

Também é possível usar o href para linkar a outra página do seu site, dessa vez optei por usar target = "_self" pois não está indo a um site externo, e na página alvo, colocarei um link para votar para a esta. Exemplo link 2
